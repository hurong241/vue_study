这是一个vue单组件的例子，包括5个文件：index.html,main.js,以及三个组件App.vue,School.vue,Student.vue
在App.vue中用到School.vue和Student.vue两个组件
当这些工作准备完成后，并不能直接解析成功，需要安装vue脚手架(vue-cli)
安装脚手架前需要先安装npm,它需要先安装node.js,安装参考安装npm
vue-cli网址:https://cli.vuejs.org/zh/
安装步骤如下：
1、下载并安装node.js
2、设置淘宝镜像：npm config set registry https://registry.npm.taobao.org
3、安装vue-cli:npm install -g @vue/cli
4、如果有提交则按提示更新，如：npm install -g npm@10.2.5
5、检查：关掉命令行工具，重新打开，输入vue如果未报错则安装成功
5、切换到要创建项目的目录(如:F:\windows_linux\github_vue_study)，
创建项目,如：vue create vue2-cli
根据要使用的vue版本选择,这里使用的vue2，故选择Default ([Vue 2] babel eslint),eslint用于语法检查
6、进入项目目录：下载完成后，根据提示进行项目目录，如：cd vue2-cli
7、启动：npm run serve
8、访问：根据提示访问对应网址,一个为本地，如:http://localhost:8080/，一个为局域网地址,如：http://192.168.181.101:8080/
